    class Subsystem

    Schema Information

    Table name: subsystems

    id         :integer(38)     not null, primary key
name       :string(255)     not null
created_at :datetime        not null
updated_at :datetime        not null

    This model represents a subsystem of the laser such as PreAmplifier or MainAmplifier. An Instance refers to it. Subsystems are normally used as filter creteria for an Instance.

    Validations

    A subsystem cannot be deleted if an Instance still refers to it.
