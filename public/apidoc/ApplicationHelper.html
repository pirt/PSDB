    formatDate(date, options={}) click to toggle source

    Format dates from the datebase to nicely formatted local time (database times are UTC). Supported options are

    :dateOnly=>true

    display only the date without the time

    :showSeconds=>false

    display without the “seconds” field

    # File app/helpers/application_helper.rb, line 21
def formatDate(date, options={})
  if (options[:dateOnly]==true)
    return date.localtime().strftime("%d.%m.%Y")
  else
    if (options[:showSeconds]==false)
      return date.localtime().strftime("%d.%m.%Y %H:%M")
    else
      return date.localtime().strftime("%d.%m.%Y %X")
    end
  end
end

def pageTitle
  base_title = "PHELIX shot database"
  if @pageTitle.nil?
    base_title
  else
    "#{base_title} | #{@pageTitle}"
  end
end
